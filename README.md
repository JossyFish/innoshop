Для запуска рекомендуется использовать Docker. 

Бд создатся сама и заполнится seed данными.

Для запуска следует в файл "\innoShop\backend\UserService\UserService.API\appsettings.json" ввести строку подключения к базе данных PostgreSQL, 
  а также данные EmailOptions, котрые представляют собой данные google  в виде пароля приложений, для этого надо создать пароль приложения 
  с названием InnoShop, в следствии чего будет сгенерирован пароль который нужно вставить в Password в appsettings.json, а также вставить
  адрес своей электронной почты в Username и FromEmail.

В файл "\innoShop\backend\ProductService\ProductService.API\appsettings.json" следует ввести строку подключения к базе данных PostgreSQL. 

При запуске через Docker в docker-compose.yml следует прописать подключения к бд в двух сервисах.

Аунтефикация/Авторизация работает за счет куки. Тестировать можно как и через фронт енд, так и с swagger. 
При подтверждении почты кроме кода можно использовать и ссылку в письме, которое приходит на почту, но так как предполагается, что приложение 
  будет использоваться через frontend, то ссылка ведет туда.

Для фронт енда используется фреймворк Next.js. Функции реализованы только для одной роли - пользователь.

У пользователей, которые создаются в seed данных, пароль от аккаунта является: admin
